# AnythingLLM (Custom Fork)

This is a modified fork of [AnythingLLM by Mintplex-Labs](https://github.com/Mintplex-Labs/anything-llm).

## Custom Features Added in This Version:

- 📱 **AnythingLLM Mobile App**: Added ability to connect to workspaces via the new mobile app.
- 📅 **Historical Timeline**: Added new historical timeline feature with PDF deep-linking.
- 📊 **XLSX Support**: Spreadsheets can now be read in chat by combining sheets.
- 🐳 **Docker Model Runner**: Added support for streaming and downloading models.
- ☁️ **AWS Bedrock**: Added streaming support for AWS Bedrock agents.
- 🔐 **New Login UI**: Completely redesigned the login page interface.

*For all original documentation, core features, and setup instructions, please refer to the [upstream repository](https://github.com/Mintplex-Labs/anything-llm).*
